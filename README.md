# Nova IPTV Player

Nova IPTV Player is a Windows IPTV player for Stalker and Xtream portal users. It includes a 7-day trial with no ads and no feature limitations.

## Features

- Stalker and Xtream portal support
- 7-day trial license
- No ads or feature limitations during the trial
- Live TV, VOD, and TV series browsing
- Search and filtering
- Favourites management
- Drag-and-drop favourite sorting
- EPG support where available from the provider
- Subtitle support where available from the provider
- Playback controls for play, pause, seek forward, and seek backward
- Keyboard controls
- Show and hide playback controls
- Expand player to fill the window
- Full-screen playback
- Hardware decoding support
- Built-in proxy support

## Coming Soon

- M3U playlist support
- Additional platforms, including iOS

## Requirements

- .NET Runtime 8.0

## Disclaimer

Nova IPTV Player is a media player only. It does not provide IPTV subscriptions, channels, streams, playlists, portal access, or any other media content.

You are responsible for sourcing your own IPTV service and ensuring that you have the legal right to access and view any content used with this application.
